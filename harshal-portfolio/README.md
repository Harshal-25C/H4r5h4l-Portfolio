# Harshal Choudhary — Portfolio (Vite + React + JavaScript)

Plain JavaScript port of the cinematic portfolio. Zero TypeScript, zero Lovable infra.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Stack

- Vite 6 + React 18 (JavaScript / JSX)
- Tailwind CSS v4 (via @tailwindcss/vite)
- framer-motion
- lucide-react

## Deploy

- **Vercel:** `npx vercel`
- **Netlify:** drag the `dist/` folder to netlify.com
- **GitHub Pages:** `npm i -D gh-pages` then add a `deploy` script
